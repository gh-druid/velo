# 소유자 자전거 등록일
HOW IT WORKS

이렇게 씁니다

01
등록 신청
차대번호, 브랜드, 모델명, 사진을 입력하고 NFC 칩을 신청합니다.
02
칩 부착
배송된 방수 NFC 칩을 프레임 안쪽이나 원하는 부품에 부착합니다.
03
스캔으로 인증
누구든 스마트폰으로 칩을 스캔하면 소유자 정보와 도난 여부를 즉시 확인할 수 있습니다.
04
양도도 간편하게
팔 때는 양도 코드 하나로 소유권 이전. 칩은 그대로, 정보만 바뀝니다.
FEATURES

벨로가 하는 것들

🔖
소유권 등록
차대번호와 소유자 정보를 연결합니다. 프레임뿐 아니라 스템, 체인링 등 부품도 개별 등록 가능. 다수 등록 시 할인 적용.
📡
NFC 즉시 조회
앱 설치 불필요. 스마트폰으로 칩을 스캔하면 브라우저에서 바로 소유자 페이지가 열립니다.
🚨
도난 신고
구독자만 신고 가능. 버튼 하나로 즉시 도난 상태 등록. 이후 누가 스캔해도 도난 경고가 표시됩니다.
🔍
도난품 조회
중고거래 전 칩 스캔 하나로 도난 여부 확인. 사기 피해를 사전에 차단합니다.
🤝
소유권 양도
양도 코드 생성 → 구매자가 입력하면 소유자 변경. 도난 상태에서는 양도 불가. 거래 이력 전체 보존.
🛡️
허위 신고 방지
도난 신고 시 법적 책임 동의 필수. 허위 신고 확인 시 계정 영구정지. 이의제기 기능 제공.
