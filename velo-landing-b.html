HOW IT WORKS

이렇게 씁니다

01

등록 신청

차대번호·브랜드·모델·사진을 입력하고 NFC 칩을 신청합니다.

02

칩 부착

배송된 방수 NFC 칩을 프레임 안쪽이나 원하는 부품에 부착합니다.

03

스캔으로 인증

스마트폰으로 칩을 스캔하면 소유자·도난 여부를 즉시 확인합니다.

04

양도도 간편

양도 코드 하나로 소유권 이전. 칩은 그대로, 정보만 바뀝니다.

FEATURES

벨로가 하는 것들

🔖

소유권 등록

차대번호와 소유자 정보를 연결. 부품도 개별 등록, 다수 등록 시 할인.

📡

NFC 즉시 조회

앱 설치 불필요. 칩 스캔 시 브라우저에서 바로 소유자 페이지가 열립니다.

🚨

도난 신고

구독자 한정. 버튼 하나로 즉시 도난 등록, 스캔 시 도난 경고 표시.

🔍

도난품 조회

중고거래 전 스캔으로 도난 여부 확인. 사기 피해를 사전 차단.

🤝

소유권 양도

코드 입력으로 소유자 변경. 도난 상태 양도 불가, 거래 이력 보존.

🛡️

허위 신고 방지

법적 책임 동의 필수. 허위 시 계정 영구정지, 이의제기 제공.
